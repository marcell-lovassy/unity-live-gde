Google Sheets API
-----------------

This package can optionally connect to Google Sheets through Google APIs when the user configures their own Google Cloud credentials. No Google credentials, API keys, service account key files, or private spreadsheet IDs are included with this package.

Google API usage is subject to Google's terms, quotas, permissions, and possible costs. Users are responsible for configuring their own Google Cloud project, credentials, spreadsheet permissions, quotas, and billing settings.
